This demo is used to display 3SPI+RGB LCD YM395T (driver IC ST7701S) by using RGB 16bit LCDC. Follow these steps:

1,Enable PSRAM in rtl8721dhp_intfcfg.c.
2,Replace the original src folder with this src folder.
2,make and download image.
3,you will see that RGBLCD displays red, green and blue cyclically.
